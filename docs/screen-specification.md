# 주간업무 관리 시스템 - 화면 기획서

## 1. 개요

### 1.1 서비스 목적
팀원들의 주간 업무를 체계적으로 기록하고, 개인/팀 단위의 진척도를 시각화하여 효율적인 업무 관리를 지원하는 웹 애플리케이션

### 1.2 주요 기능
- 주간보고 작성
- 개인별 업무 진척도 조회
- 팀별 업무 진척도 조회
- 실시간 채팅 (1:1, 단체)

---

## 2. 화면 구성

### 2.1 공통 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│  [로고]  주간업무 관리 시스템      [알림🔔] [프로필▼]       │
├────────────┬────────────────────────────────────────────────┤
│            │                                                │
│  사이드바   │              메인 콘텐츠 영역                  │
│            │                                                │
│  - 대시보드 │                                                │
│  - 주간보고 │                                                │
│  - 내 진척도│                                                │
│  - 팀 현황  │                                                │
│  - 설정    │                                                │
│            │                                        [💬]    │
└────────────┴────────────────────────────────────────────────┘
```

**공통 요소:**
- 상단 헤더: 로고, 알림 아이콘, 프로필 드롭다운
- 좌측 사이드바: 메뉴 네비게이션
- 우측 하단: 채팅 플로팅 버튼 (모든 화면에서 접근 가능)

| 파일 첨부 | 이미지, 문서 파일 전송 |
| 읽음 확인 | 메시지 읽음 상태 표시 |
| 알림 설정 | 채팅방별 알림 on/off |
| 검색 | 대화 내용 검색 |

## 4. 사용자 플로우

### 4.1 주간보고 작성 플로우
```
로그인 → 대시보드 → 주간보고 메뉴 → 새 보고서 작성
→ 업무 목록 추가 → 진척도 입력 → 임시저장/제출
```

### 4.2 진척도 확인 플로우
```
로그인 → 대시보드 → 내 진척도/팀 현황 메뉴
→ 기간 선택 → 상세 데이터 조회
```

### 4.3 채팅 플로우
```
채팅 버튼 클릭 → 채팅 패널 열림 → 대화 상대/그룹 선택
→ 메시지 전송 → (옵션) 파일 첨부
```

## 5. 반응형 설계

### 5.1 데스크톱 (1200px 이상)
- 사이드바 펼침 상태
- 대시보드 카드 4열 배치
- 채팅 패널 우측 하단 팝업

### 5.2 태블릿 (768px ~ 1199px)
- 사이드바 접힘 (아이콘만 표시)
- 대시보드 카드 2열 배치
- 채팅 패널 전체 너비의 40%

### 5.3 모바일 (767px 이하)
- 하단 네비게이션 바로 전환
- 대시보드 카드 1열 배치
- 채팅 전체 화면 모드

## 8. 부록

### 8.1 업무 상태 정의
- **대기**: 아직 시작하지 않은 업무
- **진행중**: 현재 작업 중인 업무
- **완료**: 작업이 완료된 업무
- **보류**: 외부 요인으로 중단된 업무

### 8.2 주요 API 엔드포인트 (예상)
```
POST   /api/reports          - 주간보고 생성
GET    /api/reports/:id      - 주간보고 조회
PUT    /api/reports/:id      - 주간보고 수정
GET    /api/progress/me      - 내 진척도 조회
GET    /api/progress/team    - 팀 진척도 조회
GET    /api/chat/rooms       - 채팅방 목록
POST   /api/chat/messages    - 메시지 전송
```
